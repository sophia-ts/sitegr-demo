# Demo for sitegr

[sitegr](https://github.com/ioniodi/sitegr) με σκοπό να μπορέσω να σας δείξω τις προσθήκες που έχω πραγματοποιήσει. Παρακάτω ακολουθεί το link του κλώνου όπου
μπορείτε να δείτε τις αλλαγές
